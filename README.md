# Syphax-Web
Web in C.

### Features
* Web server
* HTML builder (with translation support)

### Building
CMake 3.22.1 is required.

Simply run:
```bash
build.sh
```
or
```bash
build.bat
```

### Usage
Run
```bash
./syphax_web
```
then access the default ```http://127.0.0.1:8000/```

### Including it in your project
```c
#include "sw_html_builder.h"
```
